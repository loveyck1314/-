# 智忆药盒 - AI语音陪伴式智能用药助手

> 给每个独居老人一个24小时在线的AI儿女

## 项目简介

智忆药盒是一款集**用药提醒、AI语音陪伴、亲情互联**于一体的智能设备，专为独居老人设计。本项目为2026中美青年创客大赛参赛作品。

## 主要功能

- 🗣️ **智能用药提醒**：到点自动语音播报，对应药格自动弹开
- 💬 **AI情感陪伴**：支持日常聊天、新闻播报、戏曲播放
- 👨‍👩‍👧 **亲情互联**：老人语音呼叫子女，紧急情况一键SOS

## 技术架构

- 主控：ESP32-S3
- 语音：离线命令词 + 云端API
- 提醒：舵机 + 蜂鸣器 + LED
- 通信：WiFi (MQTT)
